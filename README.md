🌍 Global Sales Dashboard | Power BI Business Intelligence Project
📌 Project Overview

This project is an interactive Global Sales Dashboard developed in Microsoft Power BI using the Global Superstore dataset.

The dashboard provides a comprehensive analysis of worldwide sales performance, customer behavior, product profitability, market contribution, and regional trends. It enables business stakeholders to monitor key performance indicators (KPIs), identify profitable products, analyze customer segments, and make data-driven decisions.

🎯 Business Objective

The primary objective of this dashboard is to answer critical business questions:

Which markets generate the highest revenue?
Which customer segments contribute the most sales?
Who are the top-performing customers?
Which products generate maximum profit?
Which products are causing losses?
How do sales vary across different regions?
What is the impact of returns on overall business performance?
